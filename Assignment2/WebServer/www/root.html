You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /story -- takes in two names and creates and adventure. eg story?name1=bob&name2=fred
  • /color -- change the background and the font color of the page. eg color?bg=white&font=red or color?bg=000000&font=1FFF00
